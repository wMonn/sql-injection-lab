# SQL Injection Lab

Интерактивный учебный тренажёр для изучения SQL-инъекций и методов защиты баз данных.

Тренажёр предназначен для студентов и начинающих специалистов по информационной безопасности. Проект демонстрирует:
- Типы SQL-инъекций (Error-based, Union-based, Blind, Time-based)
- Методы эксплуатации уязвимостей
- Способы защиты (параметризованные запросы, ORM)




```bash
# Клонировать репозиторий
git clone https://github.com/ТВОЙ_АККАУНТ/sql-injection-lab.git

# Перейти в папку проекта
cd sql-injection-lab

# Установить зависимости
pip install -r requirements.txt

# Запустить приложение
python src/app.py
